# Stock Market Simulator

## Description
A **real-time stock market simulator** built in Java that allows users to buy, sell, and track stocks in a dynamic market.  
The simulator includes:

- **Real-time stock price updates** every few seconds  
- **Random market events** affecting stock prices  
- **Portfolio management** with real-time profit/loss calculation  
- **Transaction history** for all buy/sell operations  
- **ASCII trend graphs** to visualize stock price history  
- **Interactive console menu** for user operations  

This project demonstrates **Java OOP concepts**, **threads**, **user interaction**, and **console-based UI enhancements**.

---

## Features

1. **Dynamic Stock Prices** – Stocks fluctuate automatically using volatility and random market events.  
2. **Portfolio Management** – Buy/sell stocks and see profit/loss in real-time.  
3. **Transaction History** – Keeps a log of all stock transactions.  
4. **Stock Trends** – Visual representation of stock price history using ASCII bars.  
5. **Colored Output** – Highlights profit in green and loss in red in the console.
